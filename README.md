## 🚀 Overview
An **Instant Gratification** module that completely unlocks vertical movement. Climb at your own pace, get a boost when sprinting, and snap downwards instantly by looking down. Fully configurable via GameRules so you can dial in the perfect friction.

### ✨ Features
*   **Sprint Climbing**: Hold your sprint key while on a ladder to receive a configurable speed boost.
*   **Configurable Base Speed**: Adjust climbing & descending speeds independently to match your modpack's pacing.
*   **Snap Descend**: Look straight down (>45 degrees) while on a ladder to let gravity take over, sliding down significantly faster than sneaking.

## 🛠️ Usage & Configuration
All rules are configurable in-game (Requires **Dasik Library**):
- `ladderspeed.climbSpeed` (Default: 100)
- `ladderspeed.sprintMultiplier` (Default: 150)
- `ladderspeed.descendSpeed` (Default: 100)
- `ladderspeed.snapDescend` (Default: true)

## 🏗️ Building from Source
This project uses Gradle and requires Java 25.
```bash
git clone https://github.com/Rifaditya/Instant-Gratification-ladder-speed.git
cd Instant-Gratification-ladder-speed
./gradlew build
```
The compiled jar will be in `build/libs/`.
